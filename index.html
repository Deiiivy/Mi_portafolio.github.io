Proyects

Matriculas system

El proyecto de Sistema de Matrículas con Base de Datos es una aplicación web diseñada para gestionar el proceso de matrícula de estudiantes. Esta aplicación permitirá a los estudiantes proporcionar sus datos personales y académicos para inscribirse en cursos, y a los administradores llevar un registro de estas matrículas. La aplicación se desarrollará utilizando PHP como lenguaje de programación y Tailwind CSS para la interfaz de usuario.
